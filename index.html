HTML

Brief history of HTML

  • Between 1989 and 1990, English scientist Tim Berners-Lee wrote the first specification for the HTML language along with the client and server software to make it run while being employed at CERN near Geneva, Switzerland
  • HTML is a markup language and it was originally intended to enrich plain text documents with hypertext (hyperlinks to other documents)
  • HTML markup consists of tags that define HTML elements, which, nested in a strict hierarchical structure, compose the HTML document
  • HTML1 defined around 20 tags which were strictly intended for representing different types of text content and relationships between documents
  • Currently HTML features tags for text, multimedia, inputs and page structure, and also supports semantic markup

Web browsers

Web browsers

  • The first browser was released to the public in 1991
  • Mosaic was the first browser to allow images embedded in text and it became the world's first most popular browser
  • Netscape Navigator, released in 1994, was inspired by the success of the Mosaic browser
  • There was an intense period of competition between Netscape and Microsoft for dominance of the market in the 90s, which is also known as "The Browser Wars"

Modern browsers and browser support

Modern browsers and browser support

  • HTML, CSS and JavaScript are standards which get implemented by browsers differently
  • Modern browsers provide developer tools, an HTML & CSS inspector and debugging options
  • The job of a modern front-end developer is to ensure optimal browser compatibility and consistency of the user experience between browsers
